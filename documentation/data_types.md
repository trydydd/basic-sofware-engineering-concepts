# Data Types

Data types are used to classify different types of data that can be stored in variables. Some common data types are:

- Integers
- Floating-point numbers
- Booleans
- Characters
- Strings

```python
# Examples of different data types in Python
my_integer = 42
my_float = 3.14
my_boolean = True
my_char = 'A'
my_string = "Hello, World!"
```
